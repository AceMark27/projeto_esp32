Bem-vindo ao nosso Projeto de Sistema de Refrigeração com ESP32

Utilizando tecnologia avançada para manter sua temperatura sob controle.

O Projeto

Nosso sistema de refrigeração utiliza um ESP32 para controlar automaticamente a temperatura de uma caixa de isopor. Quando a temperatura registrada pelo sensor DHT11 atinge um valor específico, as ventoinhas são acionadas para resfriar a caixa. O sistema também pode ser controlado via aplicativo pela plataforma Blynk.

Protótipo 3D

Tecnologias Utilizadas

ESP32

Microcontrolador usado para automação do sistema.

DHT11

Sensor de temperatura e umidade para monitoramento.

LCD Display

Tela LCD para exibir a temperatura do projeto.

Blynk

Plataforma para controle remoto e monitoramento via aplicativo.
